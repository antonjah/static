# static

A simple HTTP server that serves static responses

## configuration

Configuring static is done with ENV variables

| parameter    | default    | description                  |
|:-------------|:-----------|:-----------------------------|
| HOST         |            | Bind hostname                |
| PORT         | 8080       | Bind port                    |
| RESPONSE     |            | Response body                |
| CONTENT_TYPE | plain/text | Response header Content-Type |
